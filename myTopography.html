Topography solution:

We can control:

  • font: by set font-family: Arial || Havana || other font (if fist doesn't exist, then browesr take next, and next. If nothing is fould, then use serif ). Font-family can be added ind body element or in any other.
  • Size of text: font-size: 10% px key words (small,big)with many unit type. Percentage are set acordingly to the original, default value. The most inportant thing is find base line size on root element on the pag.
  • Font boldnes: font-weight: normal / 100%
  • Font style italic: font-style: italic
  • Space between letters: letter-spacing: 4px
  • Word Spacing: word-spacing: 4px;
  • Shadow for text: text-shadow: 4px (horisontal ofset of shadow, how far form the text, a shadow should set) 4px (vertical ofset of the shadow, how far from bottom of the text the shadow should set) 4px (blur radious of the color) green (color)
  • Capital letters: text-transform: uppercase / lowercase
  • Underlined link: text-decoration: none. Is no good practice to set text underlined, becouse it sugest that something is a link

    • Topography for paragraphs

    • text-agligne: left (default) right / justify
    • Test indent (margines): test-indent: 25px. It can be use for any elements, not olny for paragraphs
    • Space between lines: line-height: 1.5. It can be in specyfic paragraphs, or in a base line in html element. That is for more readabilyty for users.

Hosted Font solution:

Hosted Font solution: easy one. We just Google font or Addobe TypeKit by passing link to chosen font and if browers do not have it, it will download it by a link. Link will be in a head section of the page. We use int by adding font name and font decoration to the css in font-family.

Self hosted fonts

Our font live in our server. We can download it form ex. ‘Font squirrel’ Ferder steps are:

  1. Create folder /fonts/
  2. Add all 4 files with fonts. Downloaded fonts have few version with the same fonts but with different format. It prevent browser which doesn't support some font format before useing custom font. So we provide as many font format as it is necessery and possible
  3. At the begining of a css file, where we will use font, we add @font-face for formats. @font-face is for any external font implementation.
  4. In global style, fx. body we add font-family with name of installed font
